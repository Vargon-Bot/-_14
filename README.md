
# -_14# Event Organizer — Приложение для управления мероприятиями

## 📋 Описание выполненной работы

Данный проект представляет собой десктопное приложение на **WPF (C#)** для организации и управления мероприятиями. 


### ✨ Реализованный функционал:
1. **Авторизация и регистрация пользователей** с разделением по ролям:
   - `Admin` (ID=1) — панель администратора
   - `Manager` (ID=2) — управление мероприятиями
   - `User` (ID=3) — личный кабинет гостя
2. **Панель менеджера**:
   - Просмотр списка мероприятий (название, дата, тип, место, статус)
   - Заглушка для создания нового события
3. **Работа с базой данных через Entity Framework**:
   - LINQ-запросы вместо ручных SQL
   - Автоматическая генерация классов сущностей (`Users`, `Events`, `Roles` и др.)
4. **Валидация ввода**: проверка паролей, уникальности логина, заполнения полей
5. **Стильный интерфейс**: единая цветовая схема, анимации кнопок, адаптивная вёрстка

### 🗄️ Структура базы данных `EventOrganizerDB`:
| Таблица | Описание |
|---------|----------|
| `Users` | Пользователи системы (логины, пароли, роли) |
| `Roles` | Справочник ролей (Admin, Manager, User) |
| `Events` | Мероприятия (название, дата, бюджет, статус) |
| `Vendors` | Поставщики услуг (кейтеринг, фото, транспорт) |
| `Guests` | Гости мероприятий |
| `Invitations` | Приглашения и ответы гостей |
| `BudgetItems` | Статьи расходов по мероприятиям |
| `Tasks` | Задачи для организаторов |
| `EventSchedule` | Расписание активностей |
| `Feedback` | Отзывы и оценки гостей |
